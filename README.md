# AuraDetect 🚀


AuraDetect is a lightweight, serverless, real-time object detection web application powered by TensorFlow.js. It runs entirely on your browser—no backend servers, no privacy concerns, just pure AI on the edge.

## 🌟 Features
* **Real-time Inference:** Detects objects directly in your browser using the COCO-SSD model.
* **Serverless:** Hosted on GitHub Pages, zero latency from server overhead.
* **Edge-AI:** All processing happens on your device (Client-side), making it fast and private.
* **Optimized:** Built for modern browsers with WebGL acceleration support.

## 🛠 Tech Stack
* **Framework:** TensorFlow.js
* **Model:** COCO-SSD
* **Deployment:** GitHub Pages

## 🚀 How to use
1. Open the [Live App Link](https://raman2005-bit.github.io/Aura-Detect/).
2. Grant camera permissions when prompted.
3. Start detecting objects!
